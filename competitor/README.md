## 游戏介绍

"我们想做一款不需要额外精力去玩的桌面挂机游戏——于是《指尖农场》诞生了。"

核心机制:
- 启动游戏后，日常工作中的键盘和鼠标输入会转化为作物生长和猫钓鱼
- 不需要刻意操作，正常打字工作农场就会自动发展
- 完成任务后可以查看仓库中积累的农产品
- 内置按键/点击统计，提供成就感
- 卖出作物获得金币，解锁更多植物
- 库存积累到一定量后，动物会来到农场

功能特性:
- 任务清单 + 时间追踪
- 番茄钟（可自定义时长）
- 环境音效，支持专注工作和学习
- 装饰系统: 30+ 农场装饰
- 动物好感度系统: 喂食、抚摸、换装
- 20种动物服装，带稀有度系统（需键盘敲击进化）
- Steam 成就、创意工坊、云存档、排行榜

Steam 标签: 休闲, 模拟, 农场模拟, 打字, 挂机, 2D, 俯视角, 卡通, 可爱, 放松, 独立, 教育, 钓鱼, 自然

## 设计参考要点

### 视觉风格
- 手绘卡通风格，色彩明亮温暖
- 键盘区域被草地/土壤纹理包裹，形成"农场岛"
- 键帽上直接显示作物生长状态
- 建筑群在键盘上方区域
- 动物在键帽间自由行走
- 夜间模式: 整体变暗，动物睡觉

### 交互模式
- 全局键盘监听，任意 App 打字都生效
- 每个按键对应一个农田格子
- 功能键有特殊功能（切换种子、收获等）
- 番茄钟造型为番茄形状

### 核心循环
打字 → 作物生长 → 收获 → 金币 → 解锁新作物/动物 → 继续打字
